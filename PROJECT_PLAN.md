# 신짜오골프 홈페이지 리뉴얼 계획

## 목표

- 기존 골프·아파트·풀빌라·차량 자료는 보존한다.
- 호치민 골프 상담 유입을 홈페이지의 최우선 목표로 둔다.
- 결제·장바구니·온라인 예약은 만들지 않는다.
- 모든 핵심 행동은 카카오 상담으로 연결한다.
- 향후 Codex가 블로그를 자동 발행할 수 있도록 데이터와 화면을 분리한다.

## 제작 순서

1. 공통 디자인·내비게이션·상담창
2. 메인페이지
3. 호치민 중심 골프장 목록
4. 골프장 상세페이지 공통 템플릿
5. 숙소·차량·지역별 상담 연결
6. 기본 SEO·사이트맵·robots
7. 반응형·링크·성능·접근성 검수
8. 기존 전체 자료의 단계적 이전

## 완료 기준

- 모바일 360px부터 데스크톱까지 가로 넘침이 없다.
- 모든 상담 버튼이 클릭한 서비스 정보를 상담창에 전달한다.
- 모든 페이지의 title, description, canonical, H1이 고유하다.
- 내부 링크와 이미지 경로가 모두 유효하다.
- 한 페이지에 중복 title이 없다.
- 이미지에 lazy loading, dimensions, alt가 적용된다.

## 현재 완료 상태

- 콘텐츠 44페이지, 리디렉션 포함 전체 HTML 85개 생성
- 기존 주소 41개를 새 주소로 영구 이전
- 기존 골프·아파트·풀빌라·차량 자료 유지
- 모든 페이지에 상담창과 카카오톡 직접 연결 제공
- 자동 복사 실패 시 문의 내용 표시·재복사 기능 제공
- 내부 경로, SEO 기본값, 이미지 속성, 버튼 안전성 자동 검수 통과
- 404, sitemap, robots, manifest, 보안·캐시 헤더 적용
- 골프장·상세 경로·블로그·브랜드 구조화 데이터와 전용 XG 로고 적용
- Vercel 최상단 배포 설정, 결과 폴더 지정, 도메인 교체 환경변수 적용
- Sites 프리뷰용 서버 엔트리와 호스팅 매니페스트 적용

실제 휴대전화와 운영 주소에서의 최종 화면 검수는 공개 직전에 진행한다.
