- 하노이, 다낭, 나트랑, 푸꾸옥 상담 연결
- 상품 판매·결제·장바구니 없이 상담 전환만 제공
- 골프장 11개, 아파트 4개, 풀빌라 15개, 기존 블로그 5개 유지
- 문의 요약 자동 복사, 복사 실패 안내, 재복사, 카카오톡 직접 열기 제공
- 콘텐츠 44페이지, 전체 HTML 80개, 기존 주소 41개 영구 이전
- SEO 메타, 구조화 데이터, sitemap, robots, 404, 보안·캐시 헤더 적용
- 골프장 `GolfCourse`, 상세 경로 `BreadcrumbList`, 블로그 `Article` 구조화 데이터 적용
- 홈페이지 브랜드 `Organization`·`WebSite` 구조화 데이터 적용
- 검색·모바일용 신짜오골프 XG 로고 192px·512px 적용
- sitemap 43개 주소에 정확한 최종 수정일 적용
- Open Graph·Twitter 공유 메타 보강
- Vercel 최상단 배포 설정과 `dist` 결과 폴더 자동 지정

- 배포 설정 파일의 공개 노출 제거
- 공개 직후 검수·검색 등록 안내 추가
- Sites 프리뷰용 서버 엔트리와 호스팅 매니페스트 적용
- Sites 프리뷰 배포 성공 및 핵심 경로 자동 확인
- 정적 클라이언트 결과 경로 연결 오류 수정
- 프리뷰 검수 기록과 실제 휴대전화 확인 항목 추가
- 새 블로그 글을 마크다운으로 작성하는 전용 콘텐츠 구조 추가
- 글 목록·상세·Article SEO·사이트맵·301 규칙 자동 생성
- 초안 제외·필수 메타데이터·중복 주소·내부 링크 자동 검수
